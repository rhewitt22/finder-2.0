Pre-made queries

These queries were designed to make the At-Risk Species Finder easier to work with. If you need more fine-grained control you can always create a custom query, or request a common query be added to this list. When a query is successfully executed the results will be displayed at the bottom of the page.

Endemic species (range includes only one state)

By default this query returns all species that are found in only one state. If you would like to limit your query to a specific state enter the full state name (not the abbreviation) into the input below. To retrieve any endemic species leave the field blank.


Species that occur in more than one state

By default this query returns all species that are found in more than one state. If you would like to limit your query to make sure that a specific state is included in the species range enter the state name (not the abbreviation) in the input below. To retrieve any endemic species leave the field blank.


Species that will be addressed by a Region other than the Southeast

By default this query returns all species that are found in more than one state. If you would like to limit your query to make sure that a specific state is included in the species range enter the state name (not the abbreviation) in the input below. To retrieve any endemic species leave the field blank.
